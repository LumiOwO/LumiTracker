## 功能

- [ ] 记录已打出的手牌
- [ ] 段位查询
- [ ] 伤害计算

---

- [ ] 多语言支持
- [ ] 应用内更新
- [ ] 云原神支持

## 运行环境

- 64 位 Windows 10 或 Windows 11
- [.NET 8.0 运行时](https://dotnet.microsoft.com/zh-cn/download/dotnet/8.0/runtime)
